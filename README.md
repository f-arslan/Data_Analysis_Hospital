# Data Analysis for Hospital
You know the story. Data is everywhere: texts, images, news, spreadsheets. Data determines our habits and future. The amount of data is growing by the second. Data analysis is good for any line of business. How to stay afloat in this great water of data? In this project, you will conduct a comprehensive study with pandas. Upload datasets, deal with data omissions and incorrect data filling, find the main statistical characteristics, and visualize your data.
